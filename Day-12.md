## Media Queries!
### CSS2
```html
<link rel="stylesheet" media="print" href="print.css">
```

### CSS3

```css
@media only screen and (min-width: 60em) {
    ....
}
```

* `only`
* min/max-height/width
* and
* orientation
    * `portrait`
    * `landscape`
